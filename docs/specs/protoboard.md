# ProtoBoard — Rectangle 2, Double Sided

> **Manufacturer:** DFRobot
> **Part Number:** FIT0203
> **Source:** [Core Electronics](https://core-electronics.com.au/protoboard-rectangle-2-double-sided.html) / [DFRobot](https://www.dfrobot.com/)

## Overview

A double-sided prototyping board designed for quickly prototyping stackable circuits. Features standard 0.1″ (2.54mm) hole spacing compatible with through-hole components and standard headers.

## Specifications

| Parameter | Value |
|-----------|-------|
| **Dimensions** | 58mm × 78mm (approx. 2.28″ × 3.07″) |
| **Hole spacing** | 0.1″ (2.54mm) standard pitch |
| **Hole diameter** | ~1mm |
| **Total holes** | 22 × 26 = 572 pads |
| **Circuit pattern** | Pad Per Hole (Square) |
| **Sides** | Double-sided |
| **Mounting holes** | 4× 3mm diameter |

## Features

- Standard 0.1″ (2.54mm) pitch hole spacing
- Double-sided copper pads
- Compatible with through-hole components
- Four mounting holes for mechanical fixing
- Pad-per-hole (square) circuit pattern — no connected traces, each pad is isolated
- Good quality FR4 PCB material
